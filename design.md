# Al-Madani Girls School - Design System

## Design Philosophy

### Color Palette
- **Primary Navy**: #1e3a8a (Deep, trustworthy navy blue representing academic excellence)
- **Secondary Sage**: #6b7280 (Sophisticated sage gray for balance and professionalism)
- **Accent Gold**: #d97706 (Warm gold accent for highlights and call-to-action elements)
- **Background Cream**: #fefce8 (Soft cream background for warmth and readability)
- **Text Charcoal**: #374151 (Dark charcoal for optimal text contrast)

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Accent Font**: "Source Serif Pro" - For quotes and special emphasis

### Visual Language
- **Educational Excellence**: Clean, professional design that conveys trust and academic achievement
- **Modern Femininity**: Sophisticated color choices that appeal to young women without being stereotypical
- **Accessibility First**: High contrast ratios, clear typography, and intuitive navigation
- **Community Focus**: Warm, welcoming design that reflects the school's inclusive values

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth entrance animations and micro-interactions
- **ECharts.js**: Interactive data visualizations for academic results and statistics
- **Splide.js**: Elegant image carousels and content sliders
- **Typed.js**: Dynamic typewriter effects for hero sections
- **Splitting.js**: Advanced text animations and effects

### Animation & Effects
- **Hero Typewriter**: Typewriter animation for school motto and key messages
- **Scroll Reveal**: Subtle fade-in animations as content enters viewport
- **Hover Transformations**: 3D tilt effects on cards and interactive elements
- **Gradient Text**: Animated gradient effects on important headings
- **Floating Elements**: Subtle floating animations on decorative elements

### Header Effect
- **Gradient Background**: Soft gradient overlay on hero sections
- **Parallax Scrolling**: Gentle parallax effect on hero images
- **Dynamic Navigation**: Smooth dropdown animations with proper timing

### Interactive Elements
- **Button Hover**: Color morphing and subtle shadow expansion
- **Card Interactions**: Lift effect with shadow depth on hover
- **Form Focus**: Smooth border color transitions and input highlighting
- **Navigation Active States**: Clear visual feedback for current page/section

### Layout Principles
- **Grid System**: Consistent 12-column grid for responsive design
- **White Space**: Generous spacing for readability and visual hierarchy
- **Content Flow**: Logical information architecture following user journey
- **Mobile First**: Responsive design starting from mobile breakpoints

### Imagery Style
- **Hero Images**: Professional, inspiring educational environments
- **Student Photos**: Diverse, confident young women in learning environments
- **Facility Images**: Modern, well-equipped classrooms and common areas
- **Event Photography**: Candid moments showing school community and achievements

### Component Styling
- **Navigation**: Clean, minimal design with clear hierarchy
- **Cards**: Subtle shadows with rounded corners for modern feel
- **Forms**: Clean, accessible design with clear labels and validation
- **Buttons**: Consistent styling with clear primary/secondary hierarchy
